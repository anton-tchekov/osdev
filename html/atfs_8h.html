Detailed Description

ATFS File System (ATFS = Amazing Technology File System)

Author
Anton Tchekov
Version
0.1
Date
29.05.2023
  • All integers are stored as little endian
  • All files are contiguous
  • Filename separator is a dot '.' not slash '/'
  • Filenames may only contain lowercase letters ('a' to 'z'), numbers ('0' to '9') and underscore ('_') and may not start with a number. File extensions are not supported.
  • Example path: "root.images.vacation.me_on_the_beach"

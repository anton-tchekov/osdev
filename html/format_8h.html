Formats and stores a series of characters and values in the provided character array buffer, according to the specified format string and a maximum buffer size.

Parameters
strA pointer to the character array buffer where the resulting formatted string will be stored
sizeThe maximum number of characters to be written to the buffer, including the terminating null character
formatA string that specifies how subsequent arguments are converted for output
...A variable number of additional arguments to be formatted and inserted into the resulting string, according to the format string
Returns
